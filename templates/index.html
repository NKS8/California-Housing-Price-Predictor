Supervised Machine Learning Model Comparisons and Price Prediction

Supervised Machine Learning Model(SML): SML models tested for better price prediction oh Housing Dataset such as Lasso, Ridge, Elastic Net and Random Forest regressions. This project studies how different SML models varies on House Price Predictions cities as they locate at different latitude. Data is pulled from openWeatherMap API, and dataset is built from over 500 cities around the World. Then visualizations are created by Matplotlib by Latitude vs Temperature, Humidity, Cloudiness, and Windspeed.
